# kriptografi-quiz

Program dengan AntarMuka GUI 
Menjalankan Program : 
1. Menginstall Python 
2. Mendapatkan Kode Program melalui repository GitHub 
	a. Salin URL HTTPS dari repository
	b. Membuka Command Prompt atau Terminal 
	c. Jalankan perintah berikut untuk mengunduh kode program 
    		git clone URL_REPOSITORI
    		cd nama_folder_repositori
3. Menginstall Tkinker 
4. Menjalankan Program 
	a. Membuka Command Prompt atau Terminal
	b. Menuju direktori yang menyimpan file 
	c. Menjalankan program 
    		python3 cipher_gui.py

5. Menggunakan Program 
	a. Masukkan file teks yang ingin dienkripsi atau didekripsi di area input.
	b. Masukkan kunci (minimal 12 karakter).
	c. Pilih jenis cipher yang ingin digunakan: Vigenere, Playfair, atau Hill.
	d. Tekan tombol Encrypt untuk mengenkripsi teks atau Decrypt untuk mendekripsinya.
	e. Output hasil akan ditampilkan di area output.
